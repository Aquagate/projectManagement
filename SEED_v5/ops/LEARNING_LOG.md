# LEARNING_LOG (エラーと知恵の記録)

このドキュメントは、プロジェクト中に発生した技術的なエラー、ハマりどころ、そしてそれをどう解決したかの「知恵」をストックする場所です。
AIエージェントはエラーを解決するたびに、ここを更新し、自分の（そして次に来るエージェントの）糧にしてください。

## [2026-01-31]
### Port EADDRINUSE (ポート競合)
- **事象**: `npm start` 起動時にポート（8081, 8082, 8083...）が次々と競合する。
- **原因**: 開発環境で多数のプロセスが動いている、あるいは終了していないプロセスが残存。
- **対処**: `server.js` に「自動ポート探索ロジック」を実装。`EADDRINUSE` エラー時に自動で `port + 1` して再試行するようにした。
- **知恵**: ポートを固定値で管理するのは限界がある。開発者体験（DX）を損なわないよう、空き地を勝手に探す「気の利いた」挙動をデフォルトにすべき。

### Agent Blocking (Waiting状態)
- **事象**: エージェントが `npm start` などの常駐プロセスを実行すると、出力待ちで「Waiting...」となり操作不能になる。
- **原因**: エージェントの実行環境がプロセスの終了（またはデタッチ）を待機するため。
- **対処**: サーバー起動は必ず**ユーザーによる手動実行**をルール化する。
- **知恵**: AIに「やっておいて」と頼むべきことと、人間がやるべきこと（Process Control）の境界線を明確にする。
