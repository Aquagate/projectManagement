# アカウント発行業務マニュアル

## 1. 概要
社員およびパートナーからの申請に基づき、社内システムのアカウントを発行する業務。

## 2. 承認基準
- **正社員**: 部長以上の承認が必要（申請時のSlackリンク等で確認）。
- **パートナー**: セキュリティチームの追加承認が必要。
    - ※注意: パートナーのマシンはBYODの場合があるため、VDIアカウントの発行が必要か確認すること。

## 3. 手順
1. 申請内容の確認（氏名、入社日、雇用形態）。
2. Active Directoryでのアカウント作成。
3. 初期パスワードの設定。
4. 申請者への通知（DM推奨）。
